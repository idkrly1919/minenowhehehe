# How to Use LangSearch

## Setting LangSearch as Your Default Search Engine

1. **Open Settings**
   - Click the settings icon (‚öôÔ∏è) in the navigation menu on the left side
   - Or click the "Settings" option in the main menu

2. **Navigate to Proxy Settings**
   - In the settings panel, scroll down to find the "Proxy" section
   - Look for "Search Engine" setting

3. **Select LangSearch**
   - Click on the current search engine dropdown
   - Select "LangSearch" from the list of options
   - A success toast will appear confirming the change

4. **Start Searching**
   - Go back to the homepage
   - Type your search query in the search box
   - Press Enter
   - You'll be redirected to the beautiful LangSearch interface!

## Features of LangSearch

### Time Filters
Once on the search results page, you can filter results by time:
- **All**: No time restrictions (default)
- **Past 24h**: Only results from the last day
- **Past Week**: Results from the last 7 days
- **Past Month**: Results from the last 30 days
- **Past Year**: Results from the last 365 days

### Search Results
Each result includes:
- **Title**: Clickable link to the original source
- **URL**: The web address of the source
- **Snippet**: A brief excerpt from the page
- **AI Summary**: Detailed AI-generated summary providing context and key information

### Visual Features
- **Vanta Fog Effect**: Beautiful animated fog background
- **Glassmorphism**: Modern frosted glass card designs
- **Smooth Animations**: Elegant transitions and hover effects
- **Responsive Design**: Works on desktop, tablet, and mobile

## Direct Access

You can also access LangSearch directly by visiting:
```
/search.html
```

Or with a pre-filled query:
```
/search.html?q=your+search+query
```

## Switching Back

To switch to a different search engine:
1. Open Settings
2. Go to Proxy section
3. Select a different search engine (Brave, Google, DuckDuckGo, etc.)
4. Your searches will now use the selected engine through the proxy

## Tips

- **URL Navigation**: If you type something with a dot (.) and no spaces, it will navigate directly to that URL instead of searching
- **Search Queries**: Any query with spaces or without dots will use your selected search engine
- **AI Summaries**: Look for the purple-bordered boxes below search results for AI-generated summaries
- **Home Button**: Click the home icon (üè†) in the top-right of the search page to return to the main app

## Troubleshooting

### Search not working?
- Check your internet connection
- Make sure "LangSearch" is selected in settings
- Try refreshing the page

### Vanta effect not showing?
- The fog effect requires WebGL support
- Try a different browser if it's not appearing
- It may take a moment to load on slower connections

### Results not loading?
- Check the browser console for errors
- The API key is embedded, so no configuration needed
- Make sure you're online

## Privacy Note

This is a private project with the API key embedded in the code. The searches are processed through the LangSearch API. For any privacy concerns, please review the LangSearch privacy policy at their website.
